# 🌐 موقع الويب - AI ChatBot

## استخدام الموقع

### 1. افتح الملف مباشرة
```bash
# افتح الملف في المتصفح
web/index.html
```

أو انسخ المسار الكامل وفتحه في أي متصفح

### 2. بدء المحادثة

1. **اكتب رسالتك** في حقل الإدخال
2. **اضغط Enter** أو اضغط زر "إرسال"
3. **انتظر الرد** من البرنامج

### 3. الأوامر المتاحة

| الزر | الوصف |
|------|--------|
| 🗑️ مسح | حذف كل المحادثات |
| ⬇️ تحميل | تحميل المحادثة كملف نصي |
